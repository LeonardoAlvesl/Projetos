  • Tag U utilizado para colocar o texto subrinhado.

  • Tag I utilizado para colocar o texto em italico.

  • Tag Strong utilizado para colocar o texto em negrito.

  • Tag input utilizado com type "text" para inserir um campo de digitação na pagina e utilizado type "number" para inserir um campo numerico na pagina.

  • Tag H utilizado para inserir o Titulo da pagina usando H1 utilizando H2 para inserir um subtilulo na pagina.

  • Tag P utilizada para inserir um paragrafo na pagina.

  • Tag A utilizada para ancoragem na navegação na pagina podendo utilizar HREF para inserir um link secundario.

  • Tag TARGET utilizada com _self para abrir o link na mesma aba e utilizada com _blank para abrir a pagina em uma nova aba.

  • Tag TITLE utilizada para dar o titulo em determinada pagina.

  • Tag LI utilizada junto com OL para criar uma lista ordenada ou utilizada com UL para criar uma lista não ordenada.
