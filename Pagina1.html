Atributos

Atributo "ID" utilizado para identificar o elemento no navegador.

Atributo CLASS serve para que o CSS e JAVASCRIPT selecione e acesse os elementos especificos atraves dos seletores de classe.

Atributo IMG serve para inserir uma imagem na pagina utilizando SRC para colocar o caminho da imagem podendo ser o link da imagem da internet ou da galeria utilizando o width podemos ajustar o tamanho da imagem.

Atributo STYLE utilizado para formatação no CSS.
